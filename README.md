## 自己紹介

個人でプログラミングを行っており、主にゲーム開発を通して設計・実装の経験を積んできました。<br>
CUIベースのゲーム開発を中心に、ユーザーの視認性を意識したUI設計や、状態管理・データ管理の実装を行っています。

また、業務ではGUIアプリケーションを含むテスト工程に携わっており、<br>
テスト観点の整理や結果の報告書作成を経験しています。

現在は、これらの経験をもとにポートフォリオの整備を進めています。
